# Sensor examples

The single-executable Python file that operates on Raspberry Pi 3B.

It is written in Chinese. Do not mind the naming 🥵

- **`test.py`**: 整體測試
- `balance.py`: 陀螺儀測試
- `cam_test.py`: 相機測試
- `detect_bluetooth.py`: 藍牙測試
- `microphone.py`: 麥克風測試
- `red_out_light.py`: 紅外線測試
- `turn.py`: 馬達測試
